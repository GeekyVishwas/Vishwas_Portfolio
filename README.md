## ✨ Features

- **Dynamic Theme Management:** Native system-preference detection with a manual toggle for a highly-refined Light / Dark UI.
- **Real-Time Contact System:** Direct API integrations built into the contact form allow for instant delivery of messages to Telegram without relying on email applications.
- **Visitor Analytics:** Secure asynchronous React hooks that capture and report detailed viewer insights.
- **Scroll-Reveal Animations:** Utilizes Framer Motion and intersection observers for silky smooth, performance-optimized viewport animations.
- **Filtering System:** An elegant project navigation bar that bifurcates Professional and Personal works alongside domain-specific filtering.

## 🛠️ Technology Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS (with highly customized utility variables for glassmorphism)
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons
- **Deployment:** GitHub Pages

## 💻 Local Installation & Setup

Want to run this codebase locally or build your own version? Follow the steps below:

### Prerequisites
- [Node.js](https://nodejs.org/en/) (v16.0 or higher recommended)
- `npm` or `yarn` package manager

### 1. Clone the Repository
```bash
git clone https://github.com/GeekyVishwas/Vishwas_Portfolio.git
cd Vishwas_Portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Variables
If you want the Contact Form and Visitor Analytics to work, create a `.env` file in the root directory format based on `.env.example`:
```env
VITE_TELEGRAM_BOT_TOKEN=your_bot_token
VITE_TELEGRAM_CHAT_ID=your_chat_id
VITE_ANALYTICS_BOT_TOKEN=your_bot_token
VITE_ANALYTICS_CHAT_ID=your_chat_id
```

### 4. Run Development Server
```bash
npm run dev
```
Navigate to `http://localhost:5173` to see the site running locally.
